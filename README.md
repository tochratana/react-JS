# Getting Start

## Dependencies

#### React routor

```bash
npm install react-router-dom
npm install localforage match-sorter sort-by
```

#### React flobite

```bash
npm i flowbite-react
```

#### Tailwindcss

```bash
npm install -D tailwindcss
npx tailwindcss init
```

#### Vite

```bash
npm create vite@latest
```

## Run

#### Install node_module

```bash
npm install
```

#### Run

```bash
npm run dev
```
